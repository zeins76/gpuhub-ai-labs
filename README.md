# GPUHub AI Labs

This repo collects small, focused AI/ML experiments that:

- run on a single GPU (24–32GB),
- show real numbers (time, VRAM, cost per experiment),
- and are easy to reproduce on any cloud GPU provider or your own hardware.

## Labs

- [SDXL + ComfyUI Lab](labs/sdxl-comfyui/README.md) – benchmark SDXL base vs refiner vs ControlNet on a 24GB GPU.
