# GPUHub AI Labs

This repo will collect small, focused AI/ML experiments that:

- run on a single GPU (24–32GB),
- show real numbers (time, VRAM, cost per experiment),
- and are easy to reproduce on any cloud GPU provider or your own hardware.

Right now it's a work in progress. The first lab will be about:

- SDXL + ComfyUI on a single 24GB GPU (benchmarking base vs refiner vs ControlNet).
